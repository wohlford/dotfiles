# dotfiles

Personal dotfiles for macOS with MacPorts.

## Contents

| File | Description |
| :--- | :--- |
| `.inputrc` | Readline — tab completion, bracketed paste |

## Installation

```bash
git clone https://github.com/wohlford/dotfiles.git ~/dotfiles
~/dotfiles/install.sh
```

## Dependencies

- [MacPorts](https://www.macports.org)
